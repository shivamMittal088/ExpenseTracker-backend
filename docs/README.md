# Backend Documentation

Developer notes and learnings from building the Expense Tracker backend.

## 📁 Structure

```
docs/
├── README.md               # This file — index of all notes
├── images/                 # Screenshots and diagrams
├── express-basics.md       # Express.js patterns and middleware
├── mongoose-tips.md        # MongoDB schema design and Mongoose tips
├── authentication.md       # JWT auth, HTTP-only cookies, bcrypt
├── typescript-express.md   # TypeScript integration with Express
├── logging-monitoring.md   # Structured logging with Axiom
└── CORS.md                 # CORS configuration and allowed origins
```

## 🖼️ Adding Screenshots

1. Save screenshots in the `images/` folder.
2. Reference them in docs with a relative path:
   ```markdown
   ![Description](./images/screenshot-name.png)
   ```

## ✍️ Conventions

- One file per topic.
- Use headings, code blocks, and inline screenshots to explain concepts.
- Keep notes concise — link to official docs for deep reference material.
